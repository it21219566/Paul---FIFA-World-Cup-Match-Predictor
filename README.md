# 🐙 Paul the AI Predictor

An advanced FIFA World Cup and international football match prediction system powered by Machine Learning, Dynamic Elo Ratings, Expected Goals (xG), Poisson Probability Models, and FastAPI.

Paul the AI Predictor analyzes historical international football matches and predicts:

- Match winner probabilities
- Draw probabilities
- Expected goals (xG)
- Most likely scorelines
- Team strength comparisons
- Head-to-head advantages

The project combines statistical football modeling with modern machine learning techniques to provide realistic match forecasts.

# 🤖 Machine Learning Architecture

The system uses a hybrid machine learning and statistical modeling approach.

## Stage 1 — Data Collection

Historical international football match data is collected from:

- International match results dataset
- International goalscorers dataset

These datasets contain:

- Match results
- Goals scored
- Tournament information
- Match dates
- Neutral venue indicators
- Goal scorer information

---

## Stage 2 — Feature Engineering

The model generates advanced football analytics features before training.

### Elo Rating Features

Dynamic Elo ratings are calculated chronologically.

Features:

- Home Team Elo
- Away Team Elo
- Elo Difference
- Home Advantage Bonus
- Tournament Importance Weighting

---

### Team Form Features

Rolling statistics are generated using recent matches.

Features:

| Feature | Description |
|----------|-------------|
| GF5 | Average goals scored in last 5 matches |
| GA5 | Average goals conceded in last 5 matches |
| NPG5 | Average non-penalty goals in last 5 matches |
| Win5 | Win percentage over last 5 matches |
| Win10 | Win percentage over last 10 matches |
| Rest Days | Days since previous match |

---

### Head-to-Head Features

Historical matchups between teams are analyzed.

Features:

- Number of previous meetings
- Historical win rate
- Average goal difference

---

# 🧠 Model Training

Two independent XGBoost models are trained.

## Home Goal Model

```python
XGBRegressor(
    objective="count:poisson",
    n_estimators=150,
    max_depth=4,
    learning_rate=0.05
)
```

Predicts:

```text
Expected Home Goals
```

---

## Away Goal Model

```python
XGBRegressor(
    objective="count:poisson",
    n_estimators=150,
    max_depth=4,
    learning_rate=0.05
)
```

Predicts:

```text
Expected Away Goals
```

---

# ⚽ Expected Goals (xG) Prediction

After both models predict expected goals:

```text
Home xG = λ₁
Away xG = λ₂
```

The system uses a Poisson probability distribution:

```math
P(X=x)=\frac{e^{-λ} λ^x}{x!}
```

to generate:

- Scoreline probabilities
- Match outcome probabilities
- Most likely score predictions

# 📊 Model Evaluation

The model was evaluated using matches played from **2023–Present**.

## Evaluation Results

| Metric | Score |
|----------|----------|
| Test Set Size | 3,618 Matches |
| Home Goal MAE | 1.04 |
| Away Goal MAE | 0.85 |
| Match Outcome Accuracy | 60.4% |
| Log Loss | 0.864 |

### Interpretation

- The model predicts away team goals slightly more accurately than home team goals.
- A 60.4% match outcome accuracy is competitive given the inherent uncertainty in football matches.
- Log-loss of 0.864 indicates reasonably calibrated probability estimates.

# 🚀 Installation

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/paul-ai-predictor.git
cd paul-ai-predictor
```

---

## 2. Create Virtual Environment

### Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

### Linux / Mac

```bash
python3 -m venv .venv
source .venv/bin/activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

Example requirements:

```txt
fastapi
uvicorn
xgboost
numpy
pandas
scipy
scikit-learn
matplotlib
requests
```

---

## 4. Start Backend Server

```bash
uvicorn Paul:app --reload
```

Server:

```text
http://127.0.0.1:8000
```

---

## 5. Launch Frontend

Open:

```text
index.html
```

in your browser.

Or run:

```bash
run_paul.bat
```

which:

1. Activates the virtual environment
2. Opens the frontend
3. Starts the FastAPI server
